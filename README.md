# Ancient Coin Checklist

An interactive web app for tracking ancient coin collections. Built as a Progressive Web App (PWA) that works offline.

## Features

- **Multiple Collection Categories:**
  - Roman Republic
  - Imperatorial
  - Roman Imperial (Londinium Mint)
  - Greek (Athens, Alexander, etc.)
  - Bactrian & Indo-Greek
  - Parthian
  - Byzantine
  - Marc Antony Legionary

- **Offline Support:** Works without internet once loaded
- **Local Storage:** Your collection progress saves automatically
- **Search & Filter:** Find coins quickly
- **Progress Tracking:** See completion percentage per category

## Usage

1. Clone or download this repository
2. Open `index.html` in a browser, or
3. Serve with any static file server:
   ```bash
   npx serve
   ```
